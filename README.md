##ほっかいどう保育園マップについて
北海道の保育所（認可、認可外）、幼稚園の地図です。
「さっぽろ保育園マップ」と「横浜市金沢区保育園マップ」のソースコードをもとに作成しました。

さっぽろ保育園マップ
-http://www.codeforsapporo.org/papamama/

## 利用データ・ライセンス

北海道（認可保育所、認可外保育所）、
国土数値情報（福祉施設、小学校区、中学校区、学校、鉄道）
地理院地図、OpenStreetMap、Bing Maps API
